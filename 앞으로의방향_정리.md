## 2. 오늘(2026.04.03) 확정된 설계 방향

### 게시물 구조 (새 구조)
```
기존: type 4개 중 하나 선택 (텍스트 OR 이미지 OR 파일 OR 링크)
변경: 텍스트 (항상) + 첨부파일 (선택: 이미지/파일/링크)

Post 구조:
- content: string (텍스트, 항상 필수)
- attachment?: { type, url, name }
```

### 패널 탭 구성 확정
```
할일 / 메모
공지는 NoticeArea에서만 (별도 섹션)
```

### UI/UX 설계 원칙 확정
```
레이어 패턴: position absolute + pointerEvents none
transition: 0.15s ease 전체 통일
Z-index: 패널10 / 드롭다운100 / 모달1000 / 모달본체1001 / 토스트9999
컴포넌트 상태: 기본/hover/선택/비활성/로딩/에러 6가지
터치 영역: 최소 44x44px
이모티콘 → SVG stroke 아이콘 교체
다크모드: CSS 변수 구조 준비 (Rehobot 개발 시 적용)
```

## 3. Hizzi Board에 AI 채팅 추가 (통합 계획)

### 방향 확정

- 사이드바에 AI 채팅 패널 추가

### UI 구조
```
[게시판 영역] | [AI 채팅 패널]
              ← 토글 버튼으로 열고 닫기
```

### 개발 순서
```
1단계: 히찌보드 현재 기능 완전 안정화 ← 지금
2단계: 사이드바에 AI 채팅 패널 추가
3단계: Claude API 연동 + 권한 연동
4단계: 웹 완전 안정화 확인
5단계: 모바일 PWA 최적화
```

## 4. Rehobot Plan — 독립 앱 (별도 개발)

### 개요
- **앱 이름:** Rehobot Plan
- **슬로건:** "하지 않아도 됩니다. Rehobot이 합니다."
- **목적:** AI 대화로 일정을 말하면 달력+할일 자동 정리
- **작업 경로:** D:\Dropbox\Dropbox\Rehobot\rehobot-plan

### 히찌보드와의 관계
```
히찌보드 = 팀 협업 (내부용)
Rehobot  = 개인 스케줄러 (외부 상용화)
```

### 핵심 기능
```
1. AI 대화창 (ChatGPT 스타일)
   사용자: "다음주 월요일 오후 2시 거래처 미팅"
   AI: "📅 4/6(월) 14:00 거래처 미팅 등록할까요?"
   [확인] → 달력 자동 등록

2. 달력 / 할일 목록
3. 스마트 쿼리 — "이번주 일정 요약해줘"
```

### GTM 전략
```
1단계: 히찌보드 팀이 첫 번째 세팅 케이스
2단계: 지인 세팅 (1~10명)
3단계: 콘텐츠 — "비전문가가 AI 시스템 만드는 과정" 숏폼
4단계: 바이럴 (50~100명)
```
