## 1. Hizzi Board 남은 작업

### 🔴 즉시 필요
```
1. 달력 개인/업무 구분
   - 개인: 본인만 보임
   - 업무: 전체 공개(기본) / 본인만 옵션

2. 달력 "편집" → "수정" 단어 변경

3. 연차 수정 시 시작/끝 날짜 지정 기능

4. 색상 시스템 정리 + 카테고리별 자동 색상 적용

5. 요청 아카이브 기능
   - 완료/반려/취소 후 → 아카이브 자동 이동
   - 별도 모달 (최신순 정렬)
   - 검색: 제목 / 보낸사람 / 받은사람
   - 데이터 삭제 없이 영구 보존
```

### 🟡 나중에
```
- 팀 리더 확정 플로우 (요청 2단계)
- 📱 모바일 전체 일관화 (별도 세션)
- 공지사항 작성 (패널에서 카테고리 공지 선택)
- 다른 회사 세팅
- 공통 훅 추가 적용 (useFileUpload, useIsMobile 등)
```

## 2. 오늘(2026.04.03) 확정된 설계 방향

### 공통 훅 구조 도입
```
위치: src/hooks/
원칙: 같은 로직 2곳 이상 반복 시 훅으로 분리
현재: useEscClose (전체 모달 ESC 닫기)
앞으로 추가 예정: useFileUpload, useIsMobile 등
```

### 파일 소통 방식
```
텍스트 복붙 대신 📎 첨부 버튼으로 파일 직접 올리기
2~3개씩 나눠서 올리면 용량 문제 없음
```

### 할일 요청 기능 구조 확정
```
- 요청받은 할일: 수정/삭제 불가, 완료 체크만 가능
- 완료 체크 → todoRequests status 'completed' 자동 업데이트
- 미완료 복구 → todoRequests status 'accepted'로 복귀
- FROM 태그: 왼쪽 테라코타 라인 + FROM 이름 인라인 태그
- 팀 요청: teamRequestId로 달력 중복 방지
- 달력 상세: 요청자/담당자/업무/완료여부 표시
```

### 달력 UX 확정
```
- 이벤트 클릭 시 일정추가 팝업 차단 (data-event="true" 패턴)
- 빈 셀 클릭 시만 일정추가 창 열림
- 3개까지 표시, 4개 이상이면 +n 더보기 팝업
```
