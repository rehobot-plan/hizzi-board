## [2026-04-13 세션 #11] close-session → progress.md 이동 신뢰성 불확실

- 우선순위: 중 (하네스 신뢰성 직결)
- 증상: /close-session 실행 후 인박스 항목이 progress.md로 실제 이동하는지 오너가 확신하지 못함. 처리 보고만 있고 검증 경로 없음.
- 제안 방향 (세션 중 논의 보류, 인박스 보관용):
  1. /close-session 실행 시 이동 전/후 progress.md diff를 콘솔 출력
  2. 이동된 항목별 "원본 인박스 라인 → progress 라인" 매핑표 보고
  3. 이동 실패 시 인박스에 "이동 실패" 플래그 남기고 다음 세션에서 재시도
