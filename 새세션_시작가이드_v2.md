## 자주 쓰는 명령어

```powershell
# 재빌드
cd D:\Dropbox\Dropbox\hizzi-board
Remove-Item -Recurse -Force .next
npm run build

# 배포
git add . && git commit -m "메시지" && npx vercel --prod

# 파일 교체 안 될 때
# → 파일 탐색기에서 직접 기존 파일 삭제 후 새 파일 붙여넣기
```

---

## 알려진 문제 & 해결책

| 문제 | 해결책 |
|------|--------|
| 파일 교체가 안 됨 | 파일 탐색기에서 직접 삭제 후 붙여넣기 |
| 빌드 에러 반복 | 파일 완전 새로 작성 (패치 중단) |
| 날짜 하루 밀림 | `T00:00:00` 로컬시간 사용 |
| EBUSY 오류 | `.next` 삭제 후 재빌드 |
| 컨텍스트 날아감 | `/clear` 후 이 파일 다시 첨부 |
