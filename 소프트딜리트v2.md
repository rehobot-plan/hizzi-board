# 소프트 딜리트 (Soft Delete) 기능 명세서

> 히찌보드 안정화 완료 후 구현할 기능.
> 새 세션에서 이 파일을 Claude에게 첨부하고 "이 기능 구현해줘"라고 하면 됩니다.

---

## 기능 개요

**목적:** 사용자가 삭제한 게시물을 관리자가 확인하고 완전 삭제 여부를 결정할 수 있는 시스템

**핵심 원칙:**
- 사용자 삭제 → 화면에서만 사라짐 (실제 데이터 유지)
- 관리자만 삭제된 게시물 열람 가능
- 관리자 최종 승인 후에만 완전 삭제

---

## 동작 흐름

```
[사용자] 게시물 삭제 클릭
         ↓
실제 삭제 X
isDeleted: true 로 변경
deletedAt, deletedBy 저장
         ↓
[사용자 화면] 게시물 사라짐 (정상)
[관리자 화면] 삭제된 게시물 목록에 표시

         ↓
[관리자] 삭제 목록 확인
- 내용 검토
- 개별 또는 일괄 완전 삭제 선택
         ↓
[완전 삭제] Firestore에서 실제 deleteDoc 실행
```

## 구현 상세

### 1. Firestore posts 필드 추가

```typescript
interface Post {
  // 기존 필드 유지
  id: string;
  panelId: string;
  type: 'text' | 'image' | 'link' | 'file';
  content: string;
  author: string;
  category?: string;
  visibleTo?: string[];
  createdAt: Date;

  // 새로 추가할 필드
  isDeleted?: boolean;      // 삭제 여부 (기본값 false)
  deletedAt?: Date;         // 삭제 시각
  deletedBy?: string;       // 삭제한 사람 email
}
```

### 2. postStore.ts 수정

```typescript
// 기존 deletePost → 소프트 딜리트로 변경
deletePost: async (postId: string, deletedBy: string) => {
  await updateDoc(doc(db, 'posts', postId), {
    isDeleted: true,
    deletedAt: new Date(),
    deletedBy: deletedBy,
  });
},

// 완전 삭제 (관리자 전용)
permanentDeletePost: async (postId: string) => {
  await deleteDoc(doc(db, 'posts', postId));
},

// 일괄 완전 삭제 (관리자 전용)
permanentDeleteAll: async (postIds: string[]) => {
  for (const id of postIds) {
    await deleteDoc(doc(db, 'posts', id));
  }
},
```

### 3. 게시물 필터링 수정

```typescript
// Panel.tsx, PostItem.tsx 등에서
// 일반 사용자에게는 isDeleted가 true인 게시물 숨김
const visiblePosts = posts.filter(p =>
  p.panelId === panelId && !p.isDeleted
);
```

### 4. 관리자 삭제 관리 화면 (page.tsx)

관리 모드에서 새 탭 또는 섹션으로 추가:

**표시 정보:**
- 게시물 내용 미리보기 (최대 50자)
- 원본 패널명
- 작성자 이름
- 작성일시
- 삭제한 사람 이름
- 삭제 일시

**기능:**
- 체크박스로 개별 선택
- 전체 선택/해제
- 선택 항목 완전 삭제
- 전체 일괄 완전 삭제
- 삭제 전 확인 모달 ("선택한 N개를 완전히 삭제합니다. 복구 불가.")

## UI 설계

### 관리자 모드 삭제 관리 탭

```
[관리자 모드]
  ├── 사용자 관리 (기존)
  └── 삭제된 게시물 관리 (신규)
       ├── 전체 선택 □
       ├── [일괄 완전삭제] 버튼
       └── 목록:
            □ [패널명] 내용미리보기...
              작성: 김진우 · 2026.03.31 14:20
              삭제: 김진우 · 2026.03.31 15:00
              [완전삭제] 버튼
```

### 스타일 (히찌보드 디자인 시스템 적용)

```tsx
// 삭제된 게시물 카드
background: '#FFF8F0'
border: '1px solid #EDE5DC'
// 삭제 정보 텍스트
color: '#C17B6B', fontSize: 11

// 완전삭제 버튼
border: '1px solid #C17B6B'
color: '#C17B6B'
fontSize: 10, uppercase

// 일괄삭제 버튼
background: '#2C1810'
color: '#FDF8F4'
```
